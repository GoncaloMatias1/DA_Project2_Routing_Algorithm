\hypertarget{md__r_e_a_d_m_e_autotoc_md2}{}\doxysubsection{Introduction}\label{md__r_e_a_d_m_e_autotoc_md2}
Developed for the Design of Algorithms course (Spring 2024) at the University of Porto, this project focuses on analyzing and solving the Travelling Salesperson Problem (TSP) in the context of ocean shipping and urban deliveries. It utilizes various heuristic approaches to efficiently handle complex routing problems that are otherwise intractable due to their size and constraints.\hypertarget{md__r_e_a_d_m_e_autotoc_md3}{}\doxysubsection{Team Members}\label{md__r_e_a_d_m_e_autotoc_md3}

\begin{DoxyItemize}
\item Implementation of a basic exhaustive TSP approach for small graphs.
\item Development of heuristic algorithms, including a triangular approximation approach for larger datasets.
\item Data parsing and processing from provided CSV files.
\item User-\/friendly menu interface for selecting and running different algorithmic approaches.
\item Visual and analytical presentation of algorithmic performance and efficiency.
\end{DoxyItemize}\hypertarget{md__r_e_a_d_m_e_autotoc_md5}{}\doxysubsection{Algorithmic Approach}\label{md__r_e_a_d_m_e_autotoc_md5}
The project uses a combination of backtracking for small graphs and heuristic methods for larger problems. A particular focus is on the triangular approximation heuristic, which provides a guaranteed approximation bound for certain classes of graphs.\hypertarget{md__r_e_a_d_m_e_autotoc_md6}{}\doxysubsection{Documentation}\label{md__r_e_a_d_m_e_autotoc_md6}

After loading the graphs, users can run the following algorithms\+:
\begin{DoxyItemize}
\item {\bfseries{Backtracking TSP}}\+: Finds the optimal solution for small graphs.
\item {\bfseries{Triangular Approximation}}\+: Provides a 2-\/approximation solution for the TSP.
\item {\bfseries{\mbox{\hyperlink{struct_cluster}{Cluster}} Approximation}}\+: Solves the TSP using clustering heuristics.
\item {\bfseries{TSP for Disconnected Graphs}}\+: Handles real-\/world graphs that are not fully connected.
