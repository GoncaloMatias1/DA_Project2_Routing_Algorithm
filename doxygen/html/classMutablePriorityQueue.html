Detailed Description

template<class T>
class MutablePriorityQueue< T >

A simple implementation of mutable priority queues, required by Dijkstra algorithm.

Template Parameters
TThe type of elements stored in the priority queue. The class T must have:
  • an accessible field int queueIndex;
  • operator< defined.
